## 365. Count 1 in Binary
https://www.lintcode.com/problem/count-1-in-binary

### Description
Count how many 1 in binary representation of a 32-bit integer.

### Example
Given 32, return 1

Given 5, return 2

Given 1023, return 9

### Challenge
If the integer is n bits with m 1 bits. Can you do it in O(m) time?
